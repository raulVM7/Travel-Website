# Travel Website

A responsive multi-section travel landing page with scroll animations, built to practice CSS Grid and responsive layouts.

## Built with
- HTML5
- CSS3 (Grid, Flexbox, custom properties)
- JavaScript
- ScrollReveal (scroll animations)
- Remix Icon

## Features
- Fully responsive with a mobile hamburger menu
- Sections: about, discover, blogs, journals, gallery, contact
- Smooth scroll and reveal-on-scroll effects
